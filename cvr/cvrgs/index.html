Usage (Avatars): 1. Download CVRGS_1.2.0.unitypackage 2. Drag the CVRGS prefab onto your hips and unpack it 3. Enable the triggers you want to use (Remember to enable Add AND Subtract) 4. Select the appropriate Setting Name under On Stay Trigger for each trigger 5. Resize the CVRGS object to cover your head, torso, and legs Optional: Create animations to disable the add/subtract triggers when the parameter has reached its max/min value ----- Usage (Props): 1. Add a pointer from the list above to your prop 2. Add a collider to the game object the pointer is on (set to trigger) to give it a shape 3. Set up toggles/animations to turn the pointer on and off as desired Notes for prop creators: - Stacking multiple pointers of the same type will create a more powerful effect - You can use a trigger set to one of the pointers above with "_Enabled" at the end to check if an avatar has the corresponding pointer enabled. e.g "CVRGS_Scale_Add_Enabled" - (Optional) Use the colours suggested above for consistency between props.
