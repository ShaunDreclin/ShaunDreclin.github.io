Usage (Avatars): 1. Download CVRGS_1.1.0.unitypackage 2. Drag the CVRGS prefab onto your hips 3. Resize it to cover your head, torso, and legs 4. Enable the triggers you want to use (Remember to enable Add AND Subtract) 5. Select the appropriate Setting Name under On Stay Trigger for each trigger Optional: Create animations to disable the add/subtract triggers when the parameter has reached its max/min value ----- Usage (Props): 1. Add a pointer from the list above to your prop 2. Add a collider to the game object the pointer is on (set to trigger) to give it a shape 3. Set up toggles/animations to turn the pointer on and off as desired Note: Stacking multiple pointers of the same type will create a more powerful effect Optional: Use the colours suggested above for consistency between props ----- Appreciate my work?
